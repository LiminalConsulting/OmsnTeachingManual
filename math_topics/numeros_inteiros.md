#### Representar e Comparar Números Positivos e Negativos

1. Reconhecer pares de pontos na reta numérica equidistantes da origem: um racional positivo e seu correspondente negativo (–a)

2. Identificar números simétricos: a e –a são simétricos entre si; 0 é simétrico de si próprio

3. Usar corretamente "+a" para designar o próprio número a; compreender "sinal de um número"

4. Identificar grandezas do dia a dia expressos em números positivos e negativos

5. Definir "semirreta de sentido positivo" associada a um ponto da reta numérica

6. Comparar números racionais usando a semirreta de sentido positivo

7. Reconhecer que 0 > qualquer negativo e 0 < qualquer positivo

8. Definir "valor absoluto" ou "módulo" de número a como distância à origem: |a|

9. Para positivos: maior valor absoluto = maior número; para negativos: menor valor absoluto = maior número

10. Números simétricos: mesmo valor absoluto com sinais contrários

11. Identificar conjunto dos "números inteiros relativos" (ℤ) formado por 0, números naturais e simétricos

### NÚMEROS INTEIROS - ADIÇÃO E SUBTRAÇÃO

### Conjuntos de Números Inteiros

- **Números Naturais:** N = {0, 1, 2, 3, 4, 5, …}
- **Números Inteiros:** Z = {…, -3, -2, -1, 0, 1, 2, 3, …}
- **Números Inteiros Positivos:** Z+ = {1, 2, 3, 4, 5, …}
- **Números Inteiros Negativos:** Z– = {…, -5, -4, -3, -2, –1}
- **Números Inteiros Não Positivos:** Z0– = {…, -5, -4, -3, -2, -1, 0}
- **Números Inteiros Não Negativos:** Z0+ = {0, 1, 2, 3, 4, 5, …}

### Reta Numérica, Valor Absoluto e Números Simétricos

A cada ponto da reta corresponde um único número (abcissa do ponto), com a origem no zero.

**Valor Absoluto:** Corresponde à distância do ponto à origem na reta numérica. O valor absoluto é sempre positivo ou zero.

**Exemplos:**
- |5| = 5
- |-5| = 5
- |0| = 0

**Números Simétricos:** Têm igual valor absoluto mas sinais opostos.

**Exemplo:** O simétrico de 5 é -5

### Adição Algébrica

#### Adição com Sinais Iguais
O resultado tem o mesmo sinal; somam-se os valores absolutos.

**Exemplos:**
- (+2) + (+4) = (+6)
- (-1) + (-3) = (-4)

#### Adição com Sinais Diferentes
O resultado tem o sinal da parcela de maior valor absoluto; subtraem-se os valores absolutos.

**Exemplo:**
- (-6) + (+4) = (-2)

### Propriedades da Adição

**Propriedade Comutativa:**
- 4 + 2 = 2 + 4

**Propriedade Associativa:**
- (4 + 2) + 5 = 2 + (4 + 5)

**Elemento Neutro:**
- 4 + 0 = 4

**Elemento Simétrico:**
- (+4) + (-4) = 0

### Subtração de Números Inteiros

Para subtrair números inteiros, adiciona-se ao primeiro número o simétrico do segundo número.

**Exemplos:**
- (+3) – (+5) = (+3) + (-5)
- (+3) – (-5) = (+3) + (+5)

### Expressões Numéricas

A simplificação de expressões numéricas segue regras para sinais consecutivos:
- Sinais iguais: resultado positivo
- Sinais diferentes: resultado negativo
